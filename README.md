Game Launcher 

О проекте

Проект "Game Launcher" представляет собой игровой лаунчер, разработанный на языке Python с использованием библиотеки PyGame. Данный проект выполняется в рамках учебной практики 3 курса колледжа и демонстрирует навыки разработки программного обеспечения, проектного мышления и работы с графическими интерфейсами.

Лаунчер объединяет три классические игры в единой оболочке с современным пользовательским интерфейсом. Основная цель проекта — создание полноценного приложения, которое может быть добавлено в портфолио разработчика и демонстрирует освоение новых технологий, не изучавшихся ранее в учебной программе.

Состав проекта

Проект включает следующие компоненты:

1. Главный лаунчер — центральное приложение с меню выбора игр
2. Игра "Тетрис" — классическая головоломка с падающими фигурами
3. Игра "Змейка" — аркадная игра с управлением движением змейки
4. Игра "Пинг-Понг" — спортивный симулятор с ракетками и мячом
